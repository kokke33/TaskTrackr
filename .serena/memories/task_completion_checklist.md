# TaskTrackr タスク完了時のチェックリスト

## 必須チェック項目

### 1. TypeScript型チェック
```bash
npm run check
```
- エラーがないことを確認
- 既知の問題: フォームのnull値処理で一部型エラーあり（対処法記載済み）

### 2. テスト実行
```bash
npm test                    # 全テスト実行（51件）
npm run test:unit          # ユニットテスト（26件）
npm run test:integration   # 統合テスト（25件）
```
- すべてのテストがパスすることを確認
- 新機能には対応するテストを追加

### 3. ビルド確認
```bash
npm run build
```
- ビルドエラーがないことを確認
- フロントエンド（Vite）とバックエンド（ESBuild）の両方

### 4. データベーススキーマ確認
```bash
npm run db:push
```
- スキーマ変更がある場合は適用
- マイグレーションが正常に実行されることを確認
